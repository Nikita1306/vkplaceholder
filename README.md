Репозиторий для задания на стажировку Java-разработчик VK 2024

Приложение написано с использованием Spring Boot. Для сборки приложения используется Maven

В приложении реализованы:
- Ролевая модель с 4 ролями (ROLE_ADMIN, ROLE_USERS, ROLE_ALBUMS, ROLE_POSTS)
- REST API для создания пользователей
- Базовая авторизация с генерацией JWT
- Логирование действий пользователя
- Кеширование запросов к внешнему ресурсу
- База данных для пользователей и ролей
- Проверка доступа к ресурсам по ролям
